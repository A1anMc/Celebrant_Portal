Setup Instructions
  1. Go to the Google Cloud Console
  2. Create a new project or select an existing one
  3. Enable the Gmail API for your project
  4. Create OAuth 2.0 credentials:
    • Application type: Desktop application
    • Download the credentials and save as credentials.json
    • Place the file in the application root directory
  5. The first time you run the scanner, you'll need to:
    • Authorize the application in your browser
    • Grant access to your Gmail account
    • The authorization token will be saved for future use
Scan Settings
Choose between 1 and 365 days. The scanner will look for:
  • Email addresses and phone numbers
  • Potential ceremony dates
  • Venue/location information
The scanner will:
  • Check for duplicate entries before creating new couples
  • Store relevant email information in the couple's notes
  • Set status as "Inquiry" for new couples
  • Mark names as "[Name Pending]" for manual update
